Project:
Phsar Srae — Khmer agriculture marketplace.

Page purpose:
Wholesale page is for factories, restaurants, hotels, exporters, and bulk buyers who need large quantity crops.
This is NOT small quantity buying.
Users should request bulk orders, make offers, or match with farmer groups/co-ops.

Design:
Premium B2B agriculture marketplace page.
Clean white/cream background, deep green accents, rounded cards, soft shadows, Khmer-first, professional buyer-focused.
Not dashboard. Not Fresh page.
